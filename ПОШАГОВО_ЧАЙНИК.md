# Пошаговая инструкция «сделал — иди дальше»

Всё по порядку. Выполняй шаги по одному. После каждого шага переходи к следующему.

---

## Часть А: Сайт уже открывается по адресу

**Если игра уже работает по адресу https://spindate.lemnity.ru** (настроил кто-то другой или ты раньше) — делай только это.

### Шаг А1
Открой в браузере ссылку: **https://vk.com/apps?act=manage**  
Войди в ВКонтакте, если попросит.

**Дальше → Шаг А2**

---

### Шаг А2
В списке найди приложение с ID **54511363** (или своё приложение для этой игры).  
Нажми на него.

**Дальше → Шаг А3**

---

### Шаг А3
Найди раздел **«Настройки»** и поле **«Адрес приложения»**.  
Впиши туда **ровно** (без пробелов, без слеша в конце):

```
https://spindate.lemnity.ru
```

Нажми **«Сохранить»**.

**Готово.** Открой в браузере **https://vk.com/app54511363** — должна открыться игра. Нажми «Войти через VK» и проверь. Если всё ок — на этом всё.

---

## Часть Б: Сайта ещё нет — ставим на сервер с нуля

**Если по адресу https://spindate.lemnity.ru пока ничего не открывается** — делай шаги Б1, Б2, Б3… по порядку.

Нужно: свой сервер (VPS) с доступом по SSH, домен spindate.lemnity.ru, привязанный к IP сервера.

---

### Шаг Б1. Подключись к серверу

На своём компьютере открой **Терминал** (Mac/Linux) или **PowerShell** / **cmd** (Windows).  
Введи (подставь свой логин и IP или домен сервера):

```bash
ssh root@IP_ТВОЕГО_СЕРВЕРА
```

или

```bash
ssh пользователь@spindate.lemnity.ru
```

Введи пароль, когда попросит. Увидишь приглашение вроде `root@server:~#` — значит, ты на сервере.

**Дальше → Шаг Б2**

---

### Шаг Б2. Проверь, есть ли Node.js

В том же окне на сервере введи:

```bash
node -v
```

- Если видишь что-то вроде **v18.x.x** или **v20.x.x** — всё ок. **Дальше → Шаг Б3.**
- Если пишет «команда не найдена» — установи Node.js (на Ubuntu/Debian):

```bash
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs
```

Снова введи `node -v` и убедись, что версия отображается.

**Дальше → Шаг Б3**

---

### Шаг Б3. Зайди в папку для сайтов и загрузи проект

Введи:

```bash
cd /var/www
```

Если папки нет, создай:

```bash
sudo mkdir -p /var/www
cd /var/www
```

**Как загрузить код:**

- **Если проект в Git:**  
  `sudo git clone URL_ТВОЕГО_РЕПОЗИТОРИЯ spindate`  
  затем: `cd spindate`

- **Если без Git:** на своём компьютере сделай архив папки проекта (без папок `node_modules` и `.next`), залей на сервер (через FileZilla, scp или панель хостинга), на сервере распакуй в `/var/www/spindate` и выполни:  
  `cd /var/www/spindate`

**Дальше → Шаг Б4**

---

### Шаг Б4. Создай файл с настройками

На сервере, в папке проекта (`/var/www/spindate`), выполни:

```bash
nano .env.local
```

Откроется редактор. Вставь **две строки** (можно скопировать целиком):

```
NEXT_PUBLIC_VK_APP_ID=54511363
NEXT_PUBLIC_APP_URL=https://spindate.lemnity.ru
```

Сохрани: **Ctrl+O**, Enter. Выйди: **Ctrl+X**.

**Дальше → Шаг Б5**

---

### Шаг Б5. Установи зависимости и собери проект

В той же папке выполни по очереди:

```bash
npm install
```

Дождись окончания. Потом:

```bash
npm run build
```

Дождись окончания без ошибок.

**Дальше → Шаг Б6**

---

### Шаг Б6. Запусти игру (проверка)

Выполни:

```bash
npm start
```

Игра через PM2 слушает порт из `ecosystem.config.cjs` (сейчас **3001**). В браузере проверка: **http://IP_ТВОЕГО_СЕРВЕРА:3001**  
Должна открыться страница игры.  
Остановить: **Ctrl+C**.

**Дальше → Шаг Б7**

---

### Шаг Б7. Сделай так, чтобы игра работала постоянно

Останови текущий запуск (**Ctrl+C**), если ещё не остановил. Затем выполни:

```bash
sudo npm install -g pm2
pm2 start npm --name "spindate" -- start
pm2 save
pm2 startup
```

Команда `pm2 startup` выведет строку вида `sudo env PATH=...` — **скопируй её целиком и выполни** на сервере. После этого приложение будет подниматься после перезагрузки сервера.

**Дальше → Шаг Б8**

---

### Шаг Б8. Настрой Nginx (чтобы открывалось по домену)

Установи Nginx, если ещё не установлен:

```bash
sudo apt update
sudo apt install -y nginx
```

Создай конфиг сайта:

```bash
sudo nano /etc/nginx/sites-available/spindate
```

Вставь (если домен другой — замени `spindate.lemnity.ru`):

```nginx
server {
    listen 80;
    server_name spindate.lemnity.ru;
    location / {
        proxy_pass http://127.0.0.1:3001;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
```

Сохрани: **Ctrl+O**, Enter. Выйди: **Ctrl+X**.

Включи сайт и перезапусти Nginx:

```bash
sudo ln -sf /etc/nginx/sites-available/spindate /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl reload nginx
```

**Дальше → Шаг Б9**

---

### Шаг Б9. Включи HTTPS (замочек)

Установи certbot и получи бесплатный сертификат:

```bash
sudo apt install -y certbot python3-certbot-nginx
sudo certbot --nginx -d spindate.lemnity.ru
```

Введи email, согласись с условиями. Certbot сам настроит HTTPS.

Проверь в браузере: **https://spindate.lemnity.ru** — должна открываться игра.

**Дальше → Шаг Б10**

---

### Шаг Б10. Укажи адрес в настройках ВКонтакте

Открой в браузере: **https://vk.com/apps?act=manage**  
Выбери приложение **54511363** (или своё).  
В **«Настройки»** → **«Адрес приложения»** впиши:

```
https://spindate.lemnity.ru
```

Сохрани.

**Готово.** Открой **https://vk.com/app54511363** — игра должна открываться из ВК, кнопка «Войти через VK» должна работать.

---

## Если что-то пошло не так

- **Не открывается по домену** — проверь, что у домена A-запись указывает на IP сервера; что Nginx запущен: `sudo systemctl status nginx`.
- **Ошибка при `npm run build`** — пришли текст ошибки; часто не хватает Node 18+ или нужно выполнить `npm install` в папке проекта.
- **В ВК не открывается** — убедись, что в настройках приложения ВК указан ровно `https://spindate.lemnity.ru` без слеша в конце и что по этому адресу в браузере открывается игра.
