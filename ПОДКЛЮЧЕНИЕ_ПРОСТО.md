# Подключение Spindate к ВКонтакте — пошагово для начинающих

Инструкция без сложных слов: что нажать и что вписать.

---

## Что у вас должно быть

1. **Сайт игры** уже открывается по адресу **https://spindate.lemnity.ru** (его настроил вы или тот, у кого сервер).
2. **Приложение ВКонтакте** уже создано, его ID: **54483214**.

Если сайт ещё не открывается по этому адресу — сначала нужно развернуть игру на сервере (см. раздел «Размещение на сервере» в конце).

---

## Шаг 1. Зайти в настройки приложения ВКонтакте

1. Откройте в браузере: **https://vk.com/apps?act=manage**
2. Войдите в свой аккаунт ВКонтакте, если попросит.
3. Найдите в списке приложение с ID **54483214** (или то, которое вы создавали для этой игры).
4. Нажмите на него.

---

## Шаг 2. Указать адрес вашей игры

1. В открывшемся окне найдите раздел **«Настройки»** (или вкладку «Настройки»).
2. Найдите поле **«Адрес приложения»** (или «URL приложения», «Ссылка на приложение»).
3. Впишите туда **ровно так**, без пробелов в начале и в конце:

   ```
   https://spindate.lemnity.ru
   ```

4. **Не добавляйте** в конец слэш `/` и ничего после домена.
5. Нажмите **«Сохранить»** (или «Применить»).

Готово: ВКонтакте теперь знает, что ваше мини-приложение живёт по этому адресу.

---

## Шаг 3. Проверить, что всё работает

1. Откройте ВКонтакте (сайт или приложение).
2. Вставьте в строку поиска или в адресную строку ссылку на ваше приложение:

   ```
   https://vk.com/app54483214
   ```

   (Если у вас другой ID приложения — замените `54483214` на него.)

3. Должна открыться ваша игра «Целуй и знакомься».
4. Нажмите **«Войти через VK»** — должен подставиться ваш профиль (имя, аватарка).

Если игра открылась и вход через VK сработал — **подключение готово**.

---

## Частые проблемы

### Игра не открывается по ссылке из VK

- Проверьте, что в настройках приложения ВК в поле «Адрес приложения» написано именно **https://spindate.lemnity.ru** (с `https://`, без пробелов и без `/` в конце).
- Откройте в браузере **https://spindate.lemnity.ru** — страница игры должна загружаться. Если не загружается, проблема в настройке сервера/домена, а не в ВК.

### После входа ничего не происходит или ошибка

- Обновите страницу (F5 или кнопка обновления в браузере).
- Попробуйте открыть игру в мобильном приложении ВКонтакте — иногда там всё работает стабильнее.

### Хочу изменить адрес игры (другой домен)

- Поменяйте «Адрес приложения» в настройках приложения ВК на новый адрес (например, `https://другой-сайт.ru`).
- В проекте игры в файле **.env.example** (и на сервере в **.env.local**) нужно будет поменять `NEXT_PUBLIC_APP_URL` на этот же новый адрес и пересобрать приложение.

## Размещение на сервере (если сайт ещё не открывается)

Если **https://spindate.lemnity.ru** у вас пока не открывается, игру нужно сначала «выложить» на сервер и привязать к этому домену.

### Вариант А: Сервер уже есть (VPS, свой хостинг)

Пошагово: как выложить игру на свой сервер и открывать её по адресу **https://spindate.lemnity.ru**.

---

#### Что понадобится

- Доступ к серверу по **SSH** (логин, пароль или ключ — выдают хостинг/провайдер).
- На сервере установлен **Node.js** версии 18 или выше (проверка: в терминале на сервере ввести `node -v`).
- Домен **spindate.lemnity.ru** привязан к IP вашего сервера (A-запись у регистратора домена).

---

#### Шаг 1. Подключиться к серверу

Откройте терминал на своём компьютере (или программу вроде PuTTY на Windows) и подключитесь по SSH. Обычно так:

```bash
ssh root@IP_ВАШЕГО_СЕРВЕРА
```

или

```bash
ssh пользователь@spindate.lemnity.ru
```

Подставьте свой логин и IP/домен. Введите пароль, если спросит.

---

#### Шаг 2. Установить Node.js (если ещё не установлен)

Проверьте:

```bash
node -v
```

Если команда не найдена или версия старее 18, установите Node (на Ubuntu/Debian):

```bash
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs
```

Проверьте снова: `node -v` и `npm -v`.

---

#### Шаг 3. Загрузить проект на сервер

**Способ 1 — через Git** (если проект в репозитории):

```bash
cd /var/www
sudo git clone URL_ВАШЕГО_РЕПОЗИТОРИЯ spindate
cd spindate
```

**Способ 2 — через архив.** На своём компьютере в папке с проектом сделайте архив (без `node_modules` и `.next`), загрузите на сервер (через scp, FileZilla или панель хостинга), на сервере распакуйте, например:

```bash
cd /var/www
sudo unzip spindate.zip -d spindate
cd spindate
```

(Путь `/var/www` можно заменить на любую папку, куда обычно кладёте сайты.)

---

#### Шаг 4. Создать файл с настройками (.env.local)

В папке проекта на сервере выполните:

```bash
nano .env.local
```

В открывшийся редактор вставьте две строки (без кавычек):

```
NEXT_PUBLIC_VK_APP_ID=54483214
NEXT_PUBLIC_APP_URL=https://spindate.lemnity.ru
```

Сохраните: `Ctrl+O`, Enter, затем выйдите: `Ctrl+X`.

---

#### Шаг 5. Установить зависимости и собрать игру

В той же папке проекта выполните по очереди:

```bash
npm install
npm run build
```

Дождитесь окончания (может занять 1–2 минуты). Ошибок быть не должно.

---

#### Шаг 6. Запустить игру

Проверочный запуск:

```bash
npm start
```

Игра поднимется на порту **3000**. В браузере откройте `http://IP_ВАШЕГО_СЕРВЕРА:3000` — должна открыться страница игры. Остановить: `Ctrl+C`.

Чтобы игра работала постоянно (и после перезагрузки сервера), лучше запускать через **PM2**:

```bash
sudo npm install -g pm2
pm2 start npm --name "spindate" -- start
pm2 save
pm2 startup
```

Последняя команда выведет строку (типа `sudo env PATH=...`), её нужно скопировать и выполнить. После этого приложение будет подниматься при перезагрузке сервера.

---

#### Шаг 7. Настроить Nginx и HTTPS (чтобы открывалось по https://spindate.lemnity.ru)

Установите Nginx и сертификат (если ещё не стоят):

```bash
sudo apt update
sudo apt install -y nginx certbot python3-certbot-nginx
```

Создайте конфиг для сайта:

```bash
sudo nano /etc/nginx/sites-available/spindate
```

Вставьте (замените `spindate.lemnity.ru` на свой домен, если другой):

```nginx
server {
    listen 80;
    server_name spindate.lemnity.ru;
    location / {
        proxy_pass http://127.0.0.1:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
```

Сохраните (`Ctrl+O`, Enter, `Ctrl+X`). Включите сайт и перезапустите Nginx:

```bash
sudo ln -sf /etc/nginx/sites-available/spindate /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl reload nginx
```

Получите бесплатный HTTPS-сертификат:

```bash
sudo certbot --nginx -d spindate.lemnity.ru
```

Следуйте подсказкам (введите email, согласитесь с условиями). Certbot сам поправит конфиг Nginx для HTTPS.

Проверьте в браузере: **https://spindate.lemnity.ru** — должна открываться игра.

### Вариант Б: Сервера нет — хочу самый простой вариант

Можно развернуть игру бесплатно на **Vercel**:

1. Зарегистрируйтесь на **https://vercel.com** (можно через аккаунт GitHub).
2. Нажмите «Add New» → «Project» и подключите репозиторий с этой игрой (или загрузите папку проекта).
3. В настройках проекта добавьте переменные (Environment Variables):
   - `NEXT_PUBLIC_VK_APP_ID` = `54483214`
   - `NEXT_PUBLIC_APP_URL` = `https://spindate.lemnity.ru` (или адрес, который Vercel даст проекту, например `https://ваш-проект.vercel.app`)
4. Нажмите Deploy. Vercel выдаст ссылку на игру (типа `https://ваш-проект.vercel.app`).
5. Если у вас свой домен **spindate.lemnity.ru**, в настройках проекта Vercel добавьте этот домен (Add Domain) и настройте у регистратора домена перенаправление так, как пишет Vercel.

После этого в настройках приложения ВК в поле «Адрес приложения» укажите тот адрес, по которому игра реально открывается (от Vercel или ваш домен).
