# Статические ресурсы приложения (assets)

Все изображения отдаются по путям вида `/assets/...`. В коде пути задаются в `lib/assets.ts` и подставляются через функцию **`assetUrl()`**, чтобы корректно работали и на своём домене, и на GitHub Pages.

## Добавление новых ресурсов

1. Положи файл в `public/assets/`.
2. Добавь путь в `lib/assets.ts` (BOTTLE_IMAGES, FRAME_SVG или отдельная константа).
3. В компонентах используй только `assetUrl(путь)` для `src` картинок.
4. При деплое убедись, что папка `public/assets/` копируется на сервер (в GitHub Actions это делается автоматически).
