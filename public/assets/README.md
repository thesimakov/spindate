- `b_standart_v2.webp` — Классическая
- `b_lemonade_v2.webp` — Лимонад
- `b_jackdaniels_v3-20d33615-6586-4c75-923c-375c37dae0e3.webp` — Виски
- `b_champagne_v3-9fde6437-79bd-474a-bff6-6ce9a8d187b0.webp` — Шампанское
- `b_baby.webp` — Детская
- `b_vip_v2.webp` — VIP-бутылка
- `b_milk_v2.webp` — Молочная

Без этих файлов в каталоге бутылочек будут отображаться заглушки (иконка с горой).
