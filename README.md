# Spindate — мини-приложение «Целуй и знакомься»

Игра в бутылочку для знакомств во ВКонтакте (VK Mini App).

## Быстрый старт

```bash
npm install
cp .env.example .env.local   # при необходимости отредактируйте
npm run build
npm start
```
